# ONESHOT (원샷) 🎯

한 손가락으로 즐기는 중독성 다트 게임. **의존성 0** — `index.html` 더블클릭으로 바로 실행됩니다.
(HTML5 Canvas + 바닐라 JS, 빌드 도구 없음)

## 🎮 플레이

### 👉 https://yellowcornsalad.github.io/oneshot/

설치·로그인 없이 브라우저에서 바로 플레이 (PC·모바일 모두 OK). 점수는 공용 리더보드에 등록돼 모두와 순위를 겨룹니다.

## 어떻게 노나요

> **누르면** 좌우 조준이 멈추고 → **누르고 있으면** 파워 게이지가 차오르고 → **떼면** 그 높이로 다트가 날아갑니다.

- **가로 조준**: 좌우로 왕복하는 조준선을 **누르는 순간** 멈춰 가로 위치를 정합니다.
- **세로 파워**: 누르고 있는 동안 우측 게이지가 위아래로 차오릅니다. **손을 떼는 순간**의 높이가 세로 위치.
- 둘 다 정중앙에서 멈추면 → **BULLSEYE 100점 + 🔥화르륵 대폭발!**
- 입력은 **클릭 · 터치 · 스페이스바** 무엇이든 OK (누르고 떼기).

## 발(다트) 시스템 🎯

- **🎯 5발로 시작.** 한 발 던질 때마다 1개 소모.
- **정중앙 BULL을 맞혀야만 다트가 +2** 보충됩니다 → 가운데를 노릴수록 오래 살아남아요.

| 영역 | 점수 | 다트 보충 |
|---|---|---|
| 정중앙 BULL | **100** | **+2** 🎯 |
| 빨강 링 | 50 | – |
| 다크 링 | 25 | – |
| 크림 링 | 10 | – |
| 바깥 링 | 5 | – |
| 보드 밖 (MISS) | 0 | – |

- 남은 다트가 **0이 되면 게임 오버**. (좌측 상단에 큰 숫자로 표시)
- 최대 12발까지 비축 가능.

## 점수 / 난이도

- **콤보**: 연속으로 맞힐수록 점수 **배수**가 올라갑니다 (3콤보마다 ×1씩, 최대 ×12). MISS 시 콤보 초기화.
- **점점 어려워짐**: 점수가 오르면 조준선·게이지가 빨라지고(초반부터 빠릿하게), **300점부터 바람**이 불어 좌우로 다트를 밀어냅니다 (미리 표시되니 조준으로 보정하세요).
- 최고기록은 브라우저에 자동 저장(localStorage).

## 중독 포인트 (설계 의도)

- **"발이 떨어지기 전에 한 발만 더"** — 잘 맞히면 발이 늘어나는 리필 루프
- **콤보 배수**로 잘할수록 점수가 기하급수적으로 터지는 쾌감
- **BULL 적중 = 노랑 불꽃 화르륵 + 충격파 + 골드 섬광 + 묵직한 사운드**의 강한 임팩트
- **즉시 재시작 + 최고기록**, (선택) 공용 리더보드로 동료와 경쟁

## 리더보드 (선택)

Supabase REST 로 익명 닉네임 공용 순위를 제공합니다. 설정 전에는 게임은 정상 동작하고 리더보드만 "미설정"으로 표시됩니다.

1. `supabase/schema.sql` 을 Supabase 대시보드 → SQL Editor 에 붙여넣고 실행 (테이블 `dart_scores` 생성).
   - Skystack과 **같은 Supabase 프로젝트를 재사용**해도 됩니다 (테이블명이 분리돼 점수가 안 섞임).
2. `supabase-config.js` 에 프로젝트 `SUPABASE_URL` 과 `anon public` 키를 넣으세요.

## 파일 구조

```
dart/
├─ index.html          # 화면 구조 + 패널
├─ style.css           # 스타일
├─ game.js             # 게임 로직(조준·게이지·발 시스템·이펙트·사운드)
├─ supabase-config.js  # 리더보드 설정(URL/anon 키)
├─ leaderboard.js      # Supabase 연동 + 익명 닉네임
└─ supabase/schema.sql # 리더보드 테이블/정책
```

## 튜닝

`game.js` 상단 상수로 손맛을 바로 조절할 수 있어요:

- `AIM_RATE0 / POW_RATE0` — 초반 조준선·게이지 속도(난이도)
- `DARTS0` — 시작 다트 수, `DART_CAP` — 최대 비축
- `refillFor()` — 점수별 다트 보충량
- `WIND_FROM / WIND_MAX` — 바람 등장 시점·세기
